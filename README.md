# ChatAsk - AI対話型アンケートアプリ（シンプル版）

AIを活用した対話型アンケート・インタビューアプリケーション

## 特徴

- 📝 選択式質問とAI対話を組み合わせたハイブリッドアンケート
- 🤖 Google Gemini APIによる自然な対話形式のインタビュー
- ⚡ Google Formライクなシンプルで直感的なUI
- 💾 **データベース不要** - LocalStorageで完結
- 📊 JSON/CSVエクスポート機能
- 🚀 小規模アンケートに最適

## 技術スタック

- **フロントエンド**: Next.js 15, React 19, TypeScript, Tailwind CSS
- **AI**: Google Gemini API
- **データ保存**: LocalStorage（ブラウザ内）
- **デプロイ**: Vercel

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. 環境変数の設定

`.env.example`をコピーして`.env.local`を作成し、Gemini APIキーを設定してください。

```bash
cp .env.example .env.local
```

必要な環境変数:
- `GOOGLE_GEMINI_API_KEY`: Google Gemini APIキー

### 3. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 使い方

### アンケート作成

1. トップページから「作成を始める」をクリック
2. アンケートのタイトルと説明を入力
3. 質問を追加（ラジオボタン、チェックボックス、テキスト、AI対話）
4. AI対話質問では、対話の目的と最大ターン数を設定
5. 保存すると自動的にアンケートページに移動

### アンケート回答

1. アンケートURLにアクセス（保存後に自動で開きます）
2. 質問に順番に回答
3. AI対話質問では、AIと自然な会話形式でやり取り
4. すべての質問に回答後、送信

### 結果確認

1. トップページから「結果を確認」をクリック
2. 左側のリストからアンケートを選択
3. 回答データを確認
4. CSVまたはJSONでエクスポート可能

## データ管理

### LocalStorageの使用

- すべてのデータはブラウザのLocalStorageに保存されます
- データベースサーバーは不要です
- ブラウザのキャッシュをクリアするとデータが消えるので注意してください

### データのバックアップ

1. 結果ページでアンケートを選択
2. 「JSONエクスポート」をクリック
3. ダウンロードしたJSONファイルを保存

### データのインポート（将来実装予定）

現在はエクスポートのみ対応しています。

## 注意事項

### LocalStorageの制限

- ブラウザごとに約5-10MBの容量制限があります
- 大量の回答データを扱う場合は注意が必要です
- ブラウザを変えるとデータは共有されません

### 推奨される使用方法

- 小規模なアンケート（100回答以下）
- 個人やチーム内での使用
- プロトタイプやテスト用途
